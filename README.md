Um CRM com IA que recebe leads automaticamente, analisa o interesse do cliente, dá pontos de 0 a 100 e avisa o vendedor quem é prioridade baseado nos pontos. A IA analisa quem tem mais potencial de fechar negócio, o vendedor fecha a venda. Sistema integrado com banco de dados Oracle e Power BI onde ali terá relatórios diários, semanais, mensais e anuais.
