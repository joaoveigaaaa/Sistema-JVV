# JVV SISTEMAS

Um sistema de CRM inteligente que automatiza o recebimento de leads, analisa o perfil e o comportamento do cliente em tempo real, e atribui uma pontuação de **0 a 100 (*Lead Scoring*)**. O sistema prioriza automaticamente os leads com maior potencial de conversão e notifica a equipe de vendas, deixando o fechamento mais ágil e assertivo.

Toda a robustez dos dados é garantida por um ecossistema corporativo, utilizando **Oracle Database** para armazenamento seguro e **Power BI** para análises estratégicas.

## Funcionalidades Principais

* **Captação Automatizada:** Recebimento e centralização de leads de forma automática.
* **Smart Lead Scoring (IA):** Análise preditiva baseada em NLP (Processamento de Linguagem Natural) e histórico para pontuar o interesse do cliente (0 a 100).
* **Alertas de Prioridade:** Notificação imediata para o vendedor sobre os leads "quentes" (com maior potencial de fechar negócio).
* **Analytics Avançado:** Dashboards dinâmicos com relatórios diários, semanais, mensais e anuais.

## Estrutura de Relatórios (Power BI)

A integração com o Power BI fornece visões analíticas divididas em:

| Período | Foco do Relatório |

| **Diário** | Produtividade dos vendedores, taxa de resposta e leads quentes gerados no dia. |

| **Semanal** | Evolução do funil de vendas, gargalos no atendimento e acertos da IA. |

| **Mensal** | Fechamento de metas, faturamento acumulado e ROI dos canais de captação. |

| **Anual** | Análise macro de crescimento, sazonalidade de mercado e previsão de receita. |
